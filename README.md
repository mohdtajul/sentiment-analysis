# Sentiment Analysis using Machine Learning

This project predicts the sentiment of user comments as Positive, Neutral, or Negative.

## Models Used
- Support Vector Machine (SVM)
- Logistic Regression
- Naive Bayes

## Features
- Text preprocessing
- TF-IDF vectorization
- Multiple model comparison
- Flask web interface

## Tech Stack
Python, Scikit-learn, Flask, Pandas, NLTK
